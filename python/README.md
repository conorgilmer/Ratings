#Ratings in Python

## ratinggraph.py
Take the years and grades from a csv file and plot a line graph
Usage python ratinggraph.py
There is also an anaconda notebook version *ratinggraph.ipynb*

## ratingscalc.py
Read in a csv file of yearly results and calculate rating changes
Usage python ratingcalc.py

There is also an anaconda notebook version *RatingCalc.ipynb*
